Electron usage

 - Install dependencies:

```bash
npm install
```

 - Run in development (starts Vite and Electron):

```bash
npm run electron:dev
```

 - Build production app (creates `dist` then packages):

```bash
npm run electron:build
```

Notes: UI is unchanged — the Electron wrapper loads the existing built app or the Vite dev server.
